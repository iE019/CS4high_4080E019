# 數值計算
```
底下程式執行後, z之值為何?
x = 3
y = 5
z = (x + y) / 2

答:沒有數值
```

```
底下程式執行後, 其值為何?
3 / 2   答:1.5
3 // 2  答:1
-1 // 2  答:-1

51 + 122 - 33*42-(44*3)/5  答:-1239.4
3000000000*3.0  答:9000000000.0
3000000000*3  答:9000000000

int(200.2)  答:200
int(2e2) 答:200
float(200) 答:200.0
```

### 複數運算
```
3 + 2j - (4 + 4j)  答:(-1-2j)
1j*1j  答:(-1+0j)
(1 + 2j) * (3 + 4j)  答:(-5+10j)

z = 3+5j
z.real  答:3.0
z.imag  答:5.0


import cmath
cmath.sqrt(-1)

答:1j
```
